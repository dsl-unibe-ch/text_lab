# Topic Modeling

This features allows you to use state-of-the-art algorithms to extract topics from raw text data with visualisation and different output formats.

## Available topic modeling algorithms

Text Lab makes 3 different Topic Modeling frameworks available. From traditional statistical appraches to more modern neural networks approaches. 

1. **LDA:** LDA is the traditional, statistical standard for topic modeling.
2. **Top2vec:** Top2Vec is a powerful neural network approach, designed to find highly specific topic clusters without relying on manual preprocessing
3. **Bertopic:** BERTopic is a modern, highly advanced AI approach that leverages Large Language Models
